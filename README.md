# M&T Search

Search engine for movies and tv-shows.

## Deployed App

[Go checkout the app](https://m-t-search.web.app/)

## Tech Stack

**Client:** React, Context API, Material-UI, Typescript

**APIs:** TMDB, IMDB

**Deployment:** Firebase
